# Progetto_Ingegneria_del_Software

## Pacchetti da installare

I pacchetti da installare per il software vanno installati con pip e con i seguenti comandi:

```
pip install pymongo[srv]
```

```
pip install qrcode
```

```
pip install PyQt5
```

```
pip install schedule
```
